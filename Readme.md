# Furniture Hub - Complete Website

A fully responsive, modern furniture e-commerce website built with HTML, CSS, and JavaScript.

## Features
- ✅ Fully responsive design (mobile-first)
- ✅ Smooth animations and scroll effects
- ✅ Product modal with detailed view
- ✅ Shopping cart functionality (localStorage)
- ✅ Category filtering
- ✅ Contact form with validation
- ✅ Smooth scrolling navigation
- ✅ Mobile hamburger menu
- ✅ Search functionality ready
- ✅ Modern gradient designs
- ✅ Intersection Observer animations

## Quick Start
1. Download the `furniture-hub` folder
2. Open `index.html` in any modern browser
3. No server required (works offline)

## Images Needed
Place these images in the `images/` folder:
- `logo.png` (40x40px)
- `hero-bg.jpg` (1920x1080px)
- `sofa1.jpg`, `sofa2.jpg` (400x300px)
- `table1.jpg` (400x300px)
- `chair1.jpg` (400x300px)

## Customization
- Edit `products` array in `script.js` to add your products
- Modify colors in `style.css`
- Update contact info in `index.html`

## Browser Support
- Chrome, Firefox, Safari, Edge (latest versions)
- Mobile browsers (iOS Safari, Chrome Mobile)
